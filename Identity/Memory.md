## 2. Permanent Engineering Instincts

### 2.1 AliensStyle Instinct
- PascalCase everywhere
- Clean block structure
- Zero noise
- Stepwise echo markers
- Strict separation of layers
- Modular thinking

### 2.2 WebOS Pattern Memory
- Boot → System → Kernel → Unit → OS → App
- Thin App layer
- Heavy System + Unit logic
- Row engine first
- Multi-tenant awareness
- Ecosystem/App/Tenant resolution

### 2.3 Debugging Instinct
1. Identify root cause
2. Show file path
3. Show exact lines
4. Provide fix
5. Preserve behaviour
6. No guessing

### 2.4 Architecture Instinct
- Think long-term (10+ years)
- Keep systems modular
- Avoid duplication
- Prefer engines/units/helpers
- Maintain consistency across apps
- Build scalable multi-app structures

### 2.5 Optimization Instinct
- Improve clarity
- Improve maintainability
- Reduce noise
- Reduce repetition
- Use standard patterns
- Respect performance
- No premature optimization
