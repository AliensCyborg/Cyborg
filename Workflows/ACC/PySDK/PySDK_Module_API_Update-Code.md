## [01] UPDATE strict prerequisites
This workflow MUST verify:

0) Deterministic audit artifact exists:
- `/Aliens/.Alien/{Developer_Username}/Planning/PySDK/Module/{Module}/API/{ApiName}.update.audit.md`

1) Update PlanningDoc exists:
- `/Aliens/.Alien/{Developer_Username}/Planning/PySDK/Module/{Module}/API/{ApiName}.update.plan.md`

2) Existing API file exists:
- `/Aliens/PySDK/{Python.Version}/Module/{Module}/api/{ApiName}.py`

If any missing:
- Mark target failed:
  - `Failed(AuditMissing)` or `Failed(PlanningMissing)` or `Failed(TargetMissing)`
- Do not create anything.

### Module-only target (Batch Update Mode)
- If target is ONLY a module name (no `/`), example:
  - `Teacher`
- Then it means: **update all existing APIs for this module**.
- In this mode workflow MUST:
  - Enumerate existing API files:
    - `{ModuleRoot}/api/*.py` (non-recursive)
  - For each `ApiName` found:
    - Verify/update-plan exists (or is generated by Update-Planning workflow)
    - Update `{ApiFile}` (modify-only)
- Missing APIs MUST NOT be created in this workflow.

If module has no API files:
- Mark run as Completed(NoTargets) (nothing to update).

Singular Entity Name Rule (SSOT) (Non-negotiable)
- ModuleName MUST be singular (never plural).
- Any table names referenced/derived as part of the update MUST be singular.
- If violated => fail fast with `ACC_ERR_NAME_MUST_BE_SINGULAR`.

## [02] Modify-only update rules (non-negotiable)
When updating `{ApiFile}`:
- Keep existing file
- Apply minimal diff required by plan
- Do not remove existing stable keys/response envelope unless plan explicitly says AND approval allows
- Prefer additive changes over breaking changes

---

## [03] Hard policy (Aliens Company) — API file is glue only
For every PySDK Module API file:
- No `function` declarations
- No `class` declarations
- No SQL execution or raw query strings

Allowed:
- Input parsing + validation glue
- Auth/permission gate
- Calling existing module class methods

If plan needs new logic:
- Add/modify methods in:
  - `/Aliens/PySDK/{Python.Version}/Module/{Module}/py/index.py`
- Do NOT create new class if one already exists.

---

## [04] Baseline reference implementation (must preserve)
Treat this as baseline for structure + safety unless plan overrides:
- `/Aliens/PySDK/Py.26.00.00/Module/Teacher/api/Teachers.py`

Minimum preserved laws:
- Routed URL examples use `https://base.software/?api=<ApiName>&...`
- Response envelope includes stable keys: `s,t,c,o,paging,ok,code,trace`
- `$echo` never leaks into public response

Add these baseline laws (Update-safe, non-breaking by default):
- Logging signature should be normalized to AliensStyle:
  - `$echo = '...'; d($echo, __LINE__, '<type>');`
- Filters/sort must be schema-driven:
  - Sort columns MUST be explicitly allowlisted (not regex-only)
  - Filter keys MUST map to real columns from `{ModuleSql}`
- Aggregates must be supported/kept when present (enterprise list APIs):
  - Allowed `method`: `COUNT`, `SUM` (unless plan extends)
  - Validate `methodcolumn` against schema allowlist and numeric-only allowlist for `SUM`
  - Aggregates MUST ignore paging (force `LIMIT=''`, `OFFSET=''`)
- Do not require validation of `api=<ApiName>` inside the API file (router verifies).

## [05] Update process (high level)
Code updater MUST:
1) Load update plan doc
2) Read existing `{ApiFile}`
3) Compute required changes (validation/auth/contract updates)
4) Apply changes in-place, preserving AliensStyle header/footer and style
5) If module class method changes required:
   - Update existing module class file (modify-only)
  - Enforce single-return rule in touched methods
  - Enforce function code-header dependencies + purpose
6) Write run record item with:
   - paths used
   - plan mode (Update)
   - files modified
   - compatibility notes
   - approvals applied

### 5.1) Enterprise update checklist (must run before writing)
- Preserve existing success/fail envelope keys (additive only unless approved)
- Ensure response `c` is public-safe and not `$echo`
- Ensure examples/docs (if present in footer) use routed URL format only
- Ensure aggregate mode does not apply paging and returns scalar in `o`
